# Portafolio Weather Frontend - M2 🌤️

Este portafolio consiste en una aplicación de clima desarrollada para el Módulo 2.

## Tecnologías usadas
- HTML5
- Uso de Bootstrap mediante CDN (CSS3 y JavaScript).
- Repositorio público en GitHub.
- Estructura de archivos organizada.

## 🚀 Funcionalidades del Portafolio
- **Dashboard de Clima:** Visualización de condiciones meteorológicas.
- **Vistas dinámicas:** Navegación fluida entre la página principal y el detalle por ciudad.
- **Diseño Responsive:** Adaptado para una visualización óptima en dispositivos  usando **Bootstrap 5**.
- **Estructura Organizada:** Separación limpia de archivos CSS, JS e imágenes en carpetas de activos.

## 🧠 Mi Proceso de Aprendizaje
Este portafolio fue un gran desafío personal. Me tomó bastante tiempo y mucha práctica llegar a este resultado. 
- **Superación de retos:** Pasé días de mucha frustración tratando de conectar correctamente mi repositorio local con GitHub. 
- **Persistencia:** A pesar de los errores de permisos y rutas, logré dominar el flujo de trabajo de Git (`add`, `commit`, `push`).
- **Logro:** Hoy me siento mucho más segura manejando la terminal y entiendo la importancia de mantener un repositorio limpio y profesional.
